# Book Management API

A RESTful API for managing a collection of books with Keycloak authentication.

## Technologies Used

- NestJS
- MongoDB
- Prisma ORM
- Keycloak for authentication
- Docker & Docker Compose
- GitHub Actions for CI/CD

## Prerequisites

- Node.js 18+
- Docker and Docker Compose
- Git
- Postman (for testing)
